# 02-市场研究阶段审计清单

> **适用文档**: 01-用户画像、02-客户旅程地图、05-竞品分析报告等  
> **审计重点**: 数据来源、用户真实性、竞品客观性  
> **版本**: v1.0.0

---

## 审计结论判定

### PASS 标准
- [ ] 所有P0检查项通过
- [ ] 数据来源可验证
- [ ] 用户画像真实
- [ ] 竞品分析客观

### FAIL 标准
- [ ] 存在未解决的P0问题
- [ ] 关键数据无来源
- [ ] 用户画像臆想
- [ ] 竞品分析失实

---

## P0 - 阻塞项（必须解决）

### 数据来源
- [ ] **市场数据来源**: 市场规模数据是否有权威来源？
- [ ] **竞品数据来源**: 竞品信息是否标注来源和获取日期？
- [ ] **用户数据来源**: 用户画像是否有调研数据支撑？
- [ ] **数据可信度**: 是否标注了数据可信度等级？

### 用户研究
- [ ] **画像真实性**: 用户画像是否基于真实用户研究？
- [ ] **JTBD深度**: 是否深挖了用户的Jobs-to-be-Done？
- [ ] **场景完整性**: 用户场景是否真实、完整？

### 竞品分析
- [ ] **竞品选择**: 竞品选择是否合理？
- [ ] **信息来源**: 竞品信息是否来自官方渠道？
- [ ] **数据时效性**: 竞品数据是否在18个月内？

---

## P1 - 重要项（强烈建议解决）

### 市场分析
- [ ] **TAM/SAM/SOM**: 市场规模计算是否科学？
- [ ] **市场趋势**: 趋势判断是否有数据支撑？
- [ ] **市场细分**: 市场细分是否合理？

### 用户洞察
- [ ] **痛点提炼**: 用户痛点是否准确提炼？
- [ ] **动机分析**: 用户动机是否深挖？
- [ ] **需求验证**: 需求是否经过验证？

### 竞品对比
- [ ] **对比维度**: 分析维度是否全面？
- [ ] **客观性**: 对比是否客观中立？
- [ ] **差异化**: 差异化分析是否清晰？

---

## P2 - 建议项（可选优化）

- [ ] 数据可视化可优化
- [ ] 案例可补充
- [ ] 分析方法可详细

---
