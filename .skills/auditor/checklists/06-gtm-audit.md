# 06-市场推广阶段审计清单

> **适用文档**: 01-GTM策略文档、02-理想客户画像等  
> **审计重点**: ICP匹配、渠道策略、数据来源  
> **版本**: v1.0.0

---

## 审计结论判定

### PASS 标准
- [ ] 所有P0检查项通过
- [ ] ICP定义清晰
- [ ] 渠道策略可行
- [ ] 数据来源标注

### FAIL 标准
- [ ] 存在未解决的P0问题
- [ ] ICP定义模糊
- [ ] 策略不可行
- [ ] 关键数据无来源

---

## P0 - 阻塞项（必须解决）

### ICP定义
- [ ] **画像清晰**: 理想客户画像是否清晰？
- [ ] **细分合理**: 客户细分是否合理？
- [ ] **痛点匹配**: 客户痛点是否匹配产品价值？

### GTM策略
- [ ] **路径清晰**: 上市路径是否清晰？
- [ ] **策略可行**: 策略是否可执行？

### 数据来源
- [ ] **渠道数据**: 渠道成本数据是否有来源？
- [ ] **定价依据**: 定价策略是否有数据支撑？

---

## P1 - 重要项（强烈建议解决）

### 渠道策略
- [ ] **渠道选择**: 渠道选择是否合理？
- [ ] **成本评估**: 渠道成本是否评估？
- [ ] **转化路径**: 转化路径是否清晰？

### 竞争分析
- [ ] **竞品识别**: 竞争对手是否识别？
- [ ] **战卡制定**: 竞争战卡是否制定？
- [ ] **应对策略**: 应对策略是否明确？

---

## P2 - 建议项（可选优化）

- [ ] 创意可优化
- [ ] 案例可补充
