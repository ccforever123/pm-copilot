# 07-运营增长阶段审计清单

> **适用文档**: 01-北极星指标、04-AB测试分析报告等  
> **审计重点**: 指标定义、实验科学、数据质量  
> **版本**: v1.0.0

---

## 审计结论判定

### PASS 标准
- [ ] 所有P0检查项通过
- [ ] 指标定义清晰
- [ ] 实验设计科学
- [ ] 数据质量可靠

### FAIL 标准
- [ ] 存在未解决的P0问题
- [ ] 指标定义模糊
- [ ] 实验设计不科学
- [ ] 数据质量不可靠

---

## P0 - 阻塞项（必须解决）

### 指标定义
- [ ] **口径清晰**: 指标口径是否清晰？
- [ ] **计算准确**: 计算方法是否准确？
- [ ] **来源可靠**: 数据来源是否可靠？

### 实验设计
- [ ] **假设清晰**: 实验假设是否清晰？
- [ ] **变量控制**: 变量是否有效控制？
- [ ] **样本足够**: 样本量是否足够？

### 数据质量
- [ ] **准确性**: 数据准确性是否保障？
- [ ] **完整性**: 数据完整性是否检查？

---

## P1 - 重要项（强烈建议解决）

### 指标体系
- [ ] **北极星指标**: 北极星指标是否明确？
- [ ] **指标层级**: 指标层级是否合理？

### 实验细节
- [ ] **统计方法**: 统计方法是否正确？
- [ ] **显著性**: 是否达到统计显著性？

---

## P2 - 建议项（可选优化）

- [ ] 可视化可优化
- [ ] 分析方法可详细

---
