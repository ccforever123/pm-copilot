# 09-职业发展阶段审计清单

> **适用文档**: 01-PM职业发展框架、08-PM晋升顾问等  
> **审计重点**: STAR结构化、成果量化、客观性  
> **版本**: v1.0.0

---

## 审计结论判定

### PASS 标准
- [ ] 所有P0检查项通过
- [ ] STAR结构完整
- [ ] 成果可量化
- [ ] 评估客观

### FAIL 标准
- [ ] 存在未解决的P0问题
- [ ] 结构严重缺失
- [ ] 成果无法量化
- [ ] 评估主观臆断

---

## P0 - 阻塞项（必须解决）

### STAR结构
- [ ] **情境明确**: Situation是否明确？
- [ ] **任务清晰**: Task是否清晰？
- [ ] **行动具体**: Action是否具体？
- [ ] **成果量化**: Result是否量化？

### 客观性
- [ ] **事实依据**: 评估是否有事实依据？
- [ ] **避免主观**: 是否避免主观臆断？

---

## P1 - 重要项（强烈建议解决）

### 竞争力评估
- [ ] **优势识别**: 核心优势是否识别？
- [ ] **差距分析**: 能力差距是否分析？

### 发展路径
- [ ] **目标明确**: 发展目标是否明确？
- [ ] **计划可行**: 提升计划是否可行？

---

## P2 - 建议项（可选优化）

- [ ] 案例可补充
- [ ] 话术可优化

---
