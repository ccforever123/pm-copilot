# 08-通用工具阶段审计清单

> **适用文档**: 01-会议纪要模板、03-NDA保密协议等  
> **审计重点**: 合规性、完整性、实用性  
> **版本**: v1.0.0

---

## 审计结论判定

### PASS 标准
- [ ] 所有P0检查项通过
- [ ] 内容合规
- [ ] 结构完整
- [ ] 实用可行

### FAIL 标准
- [ ] 存在未解决的P0问题
- [ ] 严重不合规
- [ ] 结构严重缺失

---

## P0 - 阻塞项（必须解决）

### 合规性
- [ ] **法律合规**: 是否符合法律法规？
- [ ] **公司制度**: 是否符合公司制度？

### 完整性
- [ ] **要素完整**: 必要要素是否完整？
- [ ] **流程清晰**: 流程是否清晰可执行？

---

## P1 - 重要项（强烈建议解决）

- [ ] 格式规范
- [ ] 语言准确

---

## P2 - 建议项（可选优化）

- [ ] 模板可美化
- [ ] 示例可补充

---

## 审计输出

```markdown
## 08-通用工具审计结果

### 执行摘要
- 审计结论: [PASS/FAIL/WARNING]
- P0问题: [N]个
- P1问题: [N]个
- P2建议: [N]个

### 详细检查
| 检查项 | 状态 | 等级 | 说明 |
|--------|------|------|------|
| ... | ✅/❌ | P0/P1/P2 | ... |

### 行动项
- [ ] [P0] [问题描述] -> [解决方案]
```
