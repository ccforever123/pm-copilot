# 故事板文档

## 文档概述

故事板是一种可视化叙事工具，通过连环画形式展示用户使用产品的场景和体验旅程，帮助团队理解用户情境并发现设计机会。

---

## 使用场景

- 用户场景可视化
- 设计概念沟通
- 用户旅程展示
- 团队共识对齐

---

## 故事板绘制原则

| 原则 | 说明 |
|------|------|
| 简洁清晰 | 每格表达一个核心点 |
| 情感真实 | 反映用户真实情感 |
| 故事完整 | 有起承转合 |
| 细节真实 | 基于真实用户研究 |
| 易于理解 | 不需要过多解释 |

---

## 输出要求

1. 6格结构清晰呈现
2. 包含画面描述和对白
3. 提炼关键洞察和机会
4. 基于真实用户研究
5. 整体文档控制在2页以内
