## 文档概述

精益UX画布是一个用于UX设计思维的框架工具，帮助团队快速对齐用户、业务需求和设计方案，在早期验证设计假设。

---

## 使用场景

- 新功能UX设计
- 产品体验优化
- 设计团队协作
- 用户验证规划

## 完整示例

**示例：电商搜索功能优化**

```
┌─────────────────────────────────────────────────────────────────────────┐
│ 1. 业务问题                 │ 2. 用户问题                              │
│ • 搜索转化率低于行业平均    │ • 找不到想要的商品                       │
│ • 搜索跳出率高              │ • 搜索结果不相关                         │
│ • 搜索用户价值贡献低        │ • 不知道如何输入关键词                   │
├─────────────────────────────┼───────────────────────────────────────────┤
│ 3. 成功指标                 │ 4. 用户画像                              │
│ • 搜索转化率：3% → 5%       │ • 有明确购物目的的用户                   │
│ • 搜索跳出率：40% → 30%     │ • 习惯搜索而非分类浏览                   │
│ • 搜索满意度：70% → 85%     │ • 对价格和品质有要求                     │
├─────────────────────────────────────────────────────────────────────────┤
│ 5. 构想与假设                                                            │
│ H1: 智能联想能帮助用户更准确输入关键词                                   │
│ H2: 筛选条件前置能减少无效搜索                                          │
│ H3: 个性化排序能提高搜索结果相关性                                       │
├─────────────────────────────┬───────────────────────────────────────────┤
│ 6. MVP方案                  │ 7. 验证方法                              │
│ • 搜索联想功能              │ • 用户测试：10人可用性测试               │
│ • 基础筛选条件              │ • A/B测试：搜索转化率对比                │
│ • 搜索结果排序优化          │ • 数据分析：搜索行为追踪                 │
└─────────────────────────────┴───────────────────────────────────────────┘
```

## 输出要求

2. 假设要可验证
3. MVP范围要明确
4. 使用可视化画布展示
5. 整体文档控制在1页
