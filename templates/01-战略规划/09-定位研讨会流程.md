# 定位研讨会流程

## 文档概述

定位研讨会流程提供结构化的产品定位工作方法，帮助团队明确目标用户、核心价值和差异化优势。

---

## 使用场景

- 新产品定位
- 产品重新定位
- 定位对齐与调整
- 营销策略准备

---


#### 阶段4: 定位陈述 (20分钟)

**目标**：撰写并验证定位陈述

**定位陈述模板**：

```
对于[目标用户]，他们[需求/痛点]，
我们的[产品]是一个[产品类别]，
能够[核心价值]。
不同于[竞争对手]，
我们[差异化优势]。
```

**定位验证清单**：

| 检查项 | 问题 | 通过 |
|--------|------|------|
| 真实性 | 定位是否真实可信？ | ✓/✗ |
| 相关性 | 对目标用户有意义吗？ | ✓/✗ |
| 独特性 | 与竞品有区别吗？ | ✓/✗ |
| 可持续 | 能长期保持吗？ | ✓/✗ |
| 一致性 | 与公司战略一致吗？ | ✓/✗ |

---

## 定位陈述示例

**示例1（B2B SaaS）**：

```
对于中小企业财务团队，他们每月花费大量时间手动处理发票，
我们的智能发票是一个自动化财务工具，
能够减少60%的发票处理时间。
不同于传统财务软件，
我们有AI驱动的自动识别和审批流程。
```

**示例2（消费者应用）**：

```
对于健身初学者，他们不知道如何开始科学健身，
我们的健身App是一个个人健身教练，
能够提供定制化的训练计划和专业指导。
不同于通用健身应用，
我们专注初学者，提供从零开始的渐进式课程。
```

---

## 输出要求

1. 定位陈述清晰完整
2. 通过验证清单检查
3. 明确应用场景
4. 制定行动计划
