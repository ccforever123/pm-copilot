## 文档概述

公司研究模板帮助产品经理系统化研究目标公司，用于竞品分析、合作伙伴评估、投资决策或求职准备。

---

## 使用场景

- 竞品公司研究
- 合作伙伴尽职调查
- 投资决策支持
- 求职目标公司研究

---

## 文档模板

### 公司研究框架

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          公司研究六维度                                  │
├─────────────────────────────────────────────────────────────────────────┤
│   1. 基本信息 - 公司概况、发展历程、团队                                 │
│   2. 业务模式 - 产品/服务、收入来源、客户群体                             │
│   3. 市场地位 - 市场份额、竞争格局、差异化                               │
│   4. 财务状况 - 营收、融资、估值                                         │
│   5. 技术能力 - 技术栈、研发投入、专利                                   │
│   6. 战略方向 - 未来规划、扩张计划、风险                                 │
└─────────────────────────────────────────────────────────────────────────┘
```
