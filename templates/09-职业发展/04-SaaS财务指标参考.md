## 核心指标体系

### 一、增长指标

#### 1. 年度经常性收入 (ARR)

| 要素 | 说明 |
|------|------|
| 定义 | 年化的订阅收入总额 |
| 计算公式 | ARR = MRR × 12 |
| 行业标准 | 健康增长：年增长率 > 100%（早期）；> 30%（成熟期） |

#### 2. 月度经常性收入 (MRR)

| 要素 | 说明 |
|------|------|
| 定义 | 月度的订阅收入总额 |
| 计算公式 | MRR = Σ(客户数 × 平均月费) |
| 增长构成 | 新增MRR + 扩展MRR - 流失MRR - 收缩MRR |

**MRR变化分析**：

| MRR变化类型 | 定义 | 分析重点 |
|-------------|------|----------|
| 新增MRR | 新客户带来的MRR | 获客效率 |
| 扩展MRR | 现有客户升级/增购 | 增长能力 |
| 流失MRR | 客户取消订阅 | 流失原因 |
| 收缩MRR | 客户降级/减购 | 价格敏感度 |

---

### 二、客户指标

#### 1. 客户获取成本 (CAC)

| 要素 | 说明 |
|------|------|
| 定义 | 获取一个付费客户的平均成本 |
| 计算公式 | CAC = 销售营销费用 ÷ 新增付费客户数 |
| 行业标准 | CAC < 12个月收回成本 |

#### 2. 客户终身价值 (LTV)

| 要素 | 说明 |
|------|------|
| 定义 | 一个客户在生命周期内贡献的总收入 |
| 计算公式 | LTV = ARPU × 毛利率 ÷ 流失率 |
| 行业标准 | LTV:CAC > 3:1 |

#### 3. LTV/CAC比率

| 比率 | 评估 | 说明 |
|------|------|------|
| < 1:1 | 不健康 | 获客成本高于客户价值 |
| 1:1 - 3:1 | 一般 | 需要优化 |
| 3:1 - 5:1 | 健康 | 理想状态 |
| > 5:1 | 过高 | 可能投入不足，错过增长机会 |

#### 4. 平均每用户收入 (ARPU)

| 要素 | 说明 |
|------|------|
| 定义 | 平均每个用户贡献的收入 |
| 计算公式 | ARPU = 总收入 ÷ 用户数 |
| 行业标准 | 因产品定位而异，关键看增长趋势 |

---

### 三、留存指标

#### 1. 客户流失率 (Churn Rate)

| 要素 | 说明 |
|------|------|
| 定义 | 流失客户占总客户的比例 |
| 计算公式 | 流失率 = 流失客户数 ÷ 期初客户数 |
| 行业标准 | SMB < 3%/月；企业级 < 1%/月 |

#### 2. 收入流失率 (Revenue Churn)

| 要素 | 说明 |
|------|------|
| 定义 | 流失收入占总收入的比例 |
| 计算公式 | 收入流失率 = (流失MRR + 收缩MRR) ÷ 期初MRR |
| 行业标准 | 负流失是理想状态 |

#### 3. 净收入留存率 (NRR)

| 要素 | 说明 |
|------|------|
| 定义 | 现有客户群体的收入变化率 |
| 计算公式 | NRR = (期初MRR + 扩展MRR - 流失MRR - 收缩MRR) ÷ 期初MRR × 100% |
| 行业标准 | 优秀 > 120%；良好 > 100% |

**NRR解读**：

| NRR值 | 含义 | 状态 |
|-------|------|------|
| > 100% | 现有客户贡献增长 | 净负流失 |
| = 100% | 现有客户贡献持平 | 平衡 |
| < 100% | 现有客户贡献下降 | 需要改进 |

---

### 四、效率指标

#### 1. CAC回收周期 (CAC Payback Period)

| 要素 | 说明 |
|------|------|
| 定义 | 收回获客成本所需时间 |
| 计算公式 | CAC回收周期 = CAC ÷ (ARPU × 毛利率) |
| 行业标准 | < 12个月为健康 |

#### 2. 燃烧率 (Burn Rate)

| 要素 | 说明 |
|------|------|
| 定义 | 每月净现金消耗 |
| 计算公式 | 燃烧率 = 月支出 - 月收入 |
| 注意事项 | 需要关注跑道(Runway) = 现金余额 ÷ 燃烧率 |

#### 3. 40法则 (Rule of 40)

| 要素 | 说明 |
|------|------|
| 定义 | 衡量SaaS公司健康度的综合指标 |
| 计算公式 | 40法则 = 收入增长率% + 利润率% |
| 标准 | 结果 ≥ 40% 为健康 |

**40法则示例**：

| 增长率 | 利润率 | 合计 | 评估 |
|--------|--------|------|------|
| 50% | 0% | 50% | 健康 |
| 30% | 15% | 45% | 健康 |
| 20% | 10% | 30% | 需改进 |
| 100% | -30% | 70% | 健康(高增长期) |

## 指标看板设计

**管理层看板**：

| 指标类型 | 核心指标 | 更新频率 |
|----------|----------|----------|
| 增长 | ARR, MRR, 增长率 | 月度 |
| 效率 | LTV/CAC, NRR | 月度 |
| 健康 | 流失率, 燃烧率 | 月度 |

**运营层看板**：

| 指标类型 | 核心指标 | 更新频率 |
|----------|----------|----------|
| 获客 | 新增MRR, CAC | 周度 |
| 转化 | 试用转化率 | 周度 |
| 留存 | 月度流失率 | 月度 |

## 指标诊断指南

| 问题症状 | 可能原因 | 检查指标 | 改进方向 |
|----------|----------|----------|----------|
| 增长放缓 | 获客效率低 | CAC, 转化率 | 优化获客渠道 |
| 收入下降 | 流失严重 | 流失率, NRR | 改善留存 |
| 亏损扩大 | 成本过高 | 燃烧率, CAC回收 | 提升效率 |
| LTV下降 | 客户价值低 | ARPU, 使用深度 | 增值服务 |
