# 北极星指标文档

## 文档概述

北极星指标文档定义产品的核心指标及其支撑体系，北极星指标是反映产品核心价值交付的单一关键指标，能够指引团队聚焦于最重要的成果。

---

## 使用场景

- 产品战略目标设定
- 团队工作方向对齐
- 产品增长决策参考
- 进度跟踪与复盘

**北极星指标选择原则**：

| 原则 | 说明 | 检验问题 |
|------|------|----------|
| 反映核心价值 | 指标能否反映产品为用户创造的核心价值？ | 用户获得价值时指标是否增长？ |
| 可衡量可追踪 | 指标是否可以量化并持续追踪？ | 是否有数据支撑？ |
| 可影响可操作 | 团队的行动能否影响这个指标？ | 团队日常工作是否能推动指标？ |
| 先导性 | 指标是否能预示未来成功？ | 指标增长是否带来业务增长？ |

**北极星指标示例**：

| 产品类型 | 北极星指标示例 |
|----------|----------------|
| 社交产品 | 日活跃用户数 / 用户互动次数 |
| SaaS产品 | 活跃付费用户数 / 使用核心功能的用户数 |
| 内容产品 | 内容消费时长 / 内容完成率 |
| 交易产品 | 交易总额GMV / 完成交易用户数 |
| 工具产品 | 核心功能使用次数 / 任务完成率 |

## 北极星指标检验清单

| 检验项 | 检验问题 | 是否通过 |
|--------|----------|----------|
| 价值反映 | 用户获得价值时，北极星指标是否增长？ | ✓/✗ |
| 可衡量 | 是否有可靠的数据来源和计算方法？ | ✓/✗ |
| 可影响 | 团队的日常工作是否能直接影响这个指标？ | ✓/✗ |
| 先导性 | 北极星指标增长是否预示未来业务成功？ | ✓/✗ |
| 抗操纵 | 是否有健康指标防止指标被操纵？ | ✓/✗ |
| 易理解 | 全团队是否都能理解这个指标的意义？ | ✓/✗ |

---

## 输出要求

1. 北极星指标应该是单一的
2. 输入指标控制在3-5个核心指标
3. 指标定义清晰、可计算
4. 整体文档控制在2页以内
5. 定期（每季度）评估指标有效性
