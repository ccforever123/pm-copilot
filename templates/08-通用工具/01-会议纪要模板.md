# 会议纪要模板

## 文档概述

会议纪要文档记录会议的核心内容、决策事项和后续行动，确保会议成果得到有效传达和跟进，是会议管理的重要工具。

---

## 使用场景

- 产品评审会议
- 项目进度会议
- 决策讨论会议
- 跨部门协作会议

---

## 会议纪要最佳实践

**记录原则**：

| 原则 | 说明 |
|------|------|
| 客观记录 | 如实记录会议内容，不带个人观点 |
| 突出重点 | 聚焦决策和行动项，避免流水账 |
| 明确责任 | 每个行动项有明确的负责人和时间 |
| 及时发送 | 会后24小时内发送纪要 |
| 跟进闭环 | 定期追踪行动项完成情况 |

**常见问题避免**：

| 问题 | 正确做法 |
|------|----------|
| 记录过于详细 | 聚焦决策和行动项 |
| 没有明确时间 | 每个行动项有截止日期 |
| 责任人不清晰 | 明确标注负责人 |
| 没有发送确认 | 发送后确认关键人员已阅读 |

---

## 输出要求

1. 会后24小时内完成并发送
2. 重点记录决策和行动项
3. 每个行动项有明确负责人和时间
4. 整体文档控制在2页以内
5. 发送后确认关键人员已阅读
