# 隐私政策模板

## 文档概述

隐私政策向用户说明产品如何收集、使用、存储和保护用户个人信息，是产品合规运营的必要文档。

---

## 使用场景

- 产品上线合规
- 应用商店审核
- 用户知情告知
- 法规合规要求

---

## 文档模板

### 隐私政策

**版本号**：[版本号]
**生效日期**：[日期]
**更新日期**：[日期]

---

#### 一、引言

**[公司名称]**（以下简称"我们"）非常重视用户隐私保护。本隐私政策说明我们如何收集、使用、存储、共享和保护您的个人信息。

使用我们的产品/服务即表示您同意本隐私政策。如果您不同意本政策，请停止使用我们的产品/服务。

---

#### 二、信息收集

**我们收集的信息类型**：

| 信息类型 | 具体内容 | 收集目的 | 收集方式 |
|----------|----------|----------|----------|
| 账户信息 | 用户名、邮箱、手机号 | 注册登录、身份验证 | 您主动提供 |
| 使用信息 | 浏览记录、操作记录 | 改进服务、个性化推荐 | 自动收集 |
| 设备信息 | 设备型号、操作系统、唯一标识 | 适配服务、安全保障 | 自动收集 |
| 位置信息 | 地理位置信息 | 位置相关服务 | 经授权后收集 |
| 支付信息 | 交易记录、支付方式 | 完成交易 | 经授权后收集 |

**信息收集方式**：

| 方式 | 说明 |
|------|------|
| 您主动提供 | 注册、下单、联系客服时提供的信息 |
| 自动收集 | 通过Cookie、日志等技术自动收集的信息 |
| 第三方获取 | 从第三方平台授权获取的信息 |

---

#### 三、信息使用

**我们使用您的信息用于**：

| 使用目的 | 说明 |
|----------|------|
| 提供服务 | 账户管理、订单处理、客户服务等核心功能 |
| 改进服务 | 分析用户行为、优化产品功能、提升用户体验 |
| 个性化推荐 | 根据您的偏好提供个性化内容和推荐 |
| 安全保障 | 检测和防止欺诈、保护账户安全 |
| 合规义务 | 遵守法律法规、响应合法的政府要求 |
| 经您同意的其他用途 | 您明确同意的其他用途 |

---

#### 四、信息共享

**我们不会出售您的个人信息。**

**以下情况可能共享您的信息**：

| 共享对象 | 共享目的 | 共享信息范围 |
|----------|----------|--------------|
| 服务提供商 | 支付、物流、云服务等 | 仅限必要信息 |
| 业务合作伙伴 | 联合提供服务 | 经您授权的信息 |
| 法律要求 | 遵守法律法规 | 法律要求披露的信息 |

**我们要求接收方**：
- 仅按约定目的使用您的信息
- 采取适当的安全保护措施
- 不得向其他方转让或披露

#### 七、Cookie使用

**我们使用Cookie和类似技术用于**：

| Cookie类型 | 用途 | 可否禁用 |
|------------|------|----------|
| 必要Cookie | 维持网站正常运行 | 否 |
| 功能Cookie | 记住您的偏好设置 | 是 |
| 分析Cookie | 了解用户如何使用网站 | 是 |
| 广告Cookie | 提供相关广告 | 是 |

**如何管理Cookie**：
您可以通过浏览器设置管理Cookie。禁用某些Cookie可能影响您使用我们的服务。

#### 八、未成年人保护

我们的产品/服务主要面向成年人。如果您是未成年人：
- 请在监护人指导下使用我们的服务
- 请在监护人同意后提供个人信息
- 如我们发现收集了未成年人信息，将尽快删除

## 合规要点

| 法规 | 要求要点 |
|------|----------|
| 《个人信息保护法》 | 明示同意、最小必要、目的限制 |
| 《网络安全法》 | 数据本地化、安全保护义务 |
| GDPR(如适用) | 数据主体权利、DPO指定、跨境传输 |
| CCPA(如适用) | 消费者权利、出售信息披露 |

---

## 输出要求

1. 根据实际业务调整信息范围
2. 确保与实际数据处理行为一致
3. 重要产品建议法务审核
4. 用户易于找到和理解
5. 及时更新政策内容
